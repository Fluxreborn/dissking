## Usage / 用法

### Basic / 基础

```python
from diss_king import DissEngine

dk = DissEngine()

# 中文 - 暴躁将军模式
response = dk.attack('暴躁将军', '你这段代码写的什么垃圾')
print(response)
# 输出: "你写的是什么狗屁代码？这就是你的最高水平？"

# English - IronFist mode
response = dk.attack_en('IronFist', 'Your code is trash')
print(response)
# Output: "What the fuck did you write? Is this your best?"
```
