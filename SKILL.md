# 超级吵架王 (DissKing)

> 忍一时越想越气，退一步越想越亏。用AI拯救不开心的程序员。

## 项目简介

一个帮助嘴笨程序员在语言战斗中反击的AI工具。不追求逻辑正确，只追求念头通达。

**核心创新：锚点锁定机制** - 不管对方说什么，只攻击预设的3个痛点，绝不自证。

## 核心架构


## 快速开始

```python
from diss_king import DissEngine

# 创建引擎
dk = DissEngine()

# 选择模式，输入对方的话
response = dk.attack(
    mode="金发狮王",  # 简单粗暴模式
    target="你这段代码写的什么垃圾"
)

print(response)
# 输出: "你懂什么代码？你这辈子写过什么能用的代码？我见过很多程序员，你是最差的那个。"
```

## 锚点锁定机制

**核心逻辑：不是"回应对方的话"，而是"用对方的话当素材，输出我的攻击"**

1. **预设3个攻击锚点**：每个模式有3个永不偏离的攻击方向
2. **只提取素材词**：不分析对方逻辑，只取关键词当弹药
3. **强行拉回**：检测转移话题信号（"但是"、"其实"），无视内容，继续攻击当前锚点

### 示例

```
对方: "你这段代码写的什么垃圾"
DissKing(金发狮王): "你懂什么代码？你做出过什么成绩？"

对方: "其实我已经考虑了很多因素"  <- 试图解释/转移
DissKing: "别跟我扯这事，我就问你，你做出过什么成绩？"  <- 强行拉回锚点

对方: "但是用户的需求是这样的"  <- 继续转移  
DissKing: "别跟我扯这事，我就问你，你做出过什么成绩？"  <- 坚持攻击，绝不偏移
```
